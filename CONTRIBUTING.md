# Katkıda Bulunma Rehberi

Bu projeye katkıda bulunmak için aşağıdaki adımları izleyin:

## Katkı Türleri

- **Hata Düzeltmeleri**: Mevcut hataları düzeltme
- **Yeni Özellikler**: Uygulamaya yeni özellikler ekleme
- **İyileştirmeler**: Mevcut özellikleri geliştirme
- **Dokümantasyon**: Belgeleri güncelleme

## Geliştirme Ortamı Kurulumu

1. Depoyu fork edin
2. Repoyu klonlayın:
   ```bash
   git clone https://github.com/kullanici-adi/gubre_tavsiye.git
   ```
3. Gereksinimleri yükleyin:
   ```bash
   pip install -r requirements.txt
   ```
4. Yeni bir dal oluşturun:
   ```bash
   git checkout -b ozellik/ozellik-adi
   ```

## Kodlama Standartları

- PEP 8 stil kurallarına uyun
- Fonksiyon ve değişken isimleri için snake_case kullanın
- Sınıf isimleri için PascalCase kullanın
- Yeterli yorum ekleyin

## Pull Request Oluşturma

1. Değişikliklerinizi test edin
2. Commit mesajlarınızı açıklayıcı yazın
3. Pull request oluşturun
4. Değişikliklerinizi açıklayın

## Hata Bildirimi

Hata bildirimi için:
1. Sorunu ayrıntılı açıklayın
2. Tekrar üretim adımlarını belirtin
3. Beklenen ve gerçek davranışları karşılaştırın
4. Ekran görüntüsü veya log varsa ekleyin

## Sorularınız Var mı?

Sorularınız için geliştirici ile iletişime geçebilirsiniz.
