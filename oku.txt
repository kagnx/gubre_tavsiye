Güncellendi ve exe oluşturuldu. Şimdi çalıştırma talimatlarını vereceğim.
---
Mobil Uygulama Nasıl Çalıştırılır
# 1. Kivy ve KivyMD kur
pip install kivy kivymd
# 2. Mobil uygulamayı çalıştır
python mobile/main.py
APK oluşturmak için:
pip install buildozer
cd mobile
buildozer android debug
---
Web Versiyonu Nasıl Çalıştırılır
# 1. Flask kur
pip install flask
# 2. Web sunucusunu başlat
python web/app.py
Tarayıcıda aç:
http://localhost:5000
---
Masaüstü Uygulaması
# Python ile doğrudan
python gubre_tavsiye.py
# veya exe ile (kurulum gerektirmez)
dist\gubre_tavsiyesi.exe
